== Description ==

Newsletter PRO lets you collect subscribers from your website with a single or double opt-in subscription system.

Create and send beautiful HTML newsletters to your members and subscribers within seconds.

*Newsletter Pro Features*
1. Easily manage your newsletters.
2. FREE & Professional designed templates.
3. Track the statistics of your subscribers.	
4. Easy implementation, can be used as a widget.
5. Ability to modify subscription settings using a simple all-in one dashboard.

== Installation ==
1. Download and upload the file `newsletter-pro.zip` from the plugin panel or unzip the file and upload the folder `newsletter-pro` to the `/wp-content/plugins/` directory
2. Activate the newsletter pro plugin through the `Plugins` panel in your WordPress Site.
3. Go to WP Panel -> `Newsletter Pro` to configure the plugin.
